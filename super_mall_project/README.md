
# Super Mall Web Application

## Technologies
- HTML
- CSS
- JavaScript
- Firebase

## Features
- Manage Shop Details
- Manage Offers
- Product Listing
- Category Wise Details
- Floor Wise Details
- Firebase Ready
- Logging Support

## How to Run
1. Download the project
2. Open index.html in browser
3. Add Firebase credentials in js/firebase-config.js

## GitHub Workflow
- Push the project to a public GitHub repository
- Maintain README and commits properly
